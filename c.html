C

C is a general-purpose, imperative programming language that was developed in the early 1970s by Dennis Ritchie at Bell Labs.

C has had a profound influence on many other programming languages, including C++, Java, JavaScript, and more.

C is commonly used for system programming, embedded programming, and low-level programming where performance is crucial.
