Linguagens de programação. O que são?

Assim como existe a linguagem humana, utilizada para que seja possível a comunicação entre pessoas, há também a linguagem de programação, que permite a comunicação entre humano-máquina. Isso porque o computador não tem capacidade de compreender a linguagem utilizada por nós.

É através de regras de sintaxe e de semântica que podemos enviar dados e instruções para que o computador execute alguma tarefa. E é esse conjunto de regras que forma uma determinada linguagem de programação.

As linguagens possuem características próprias que as diferem umas das outras. Uma dessas características é a sua proximidade ou da linguagem binária de máquina (baixo nível), ou da linguagem humana (alto nível). Se não houvesse linguagens de programação de alto nível, seria necessário passar informações ao computador de forma binária, o que tornaria o trabalho extremamente propenso ao erro.

Com as linguagens de alto nível, escrevem-se algoritmos de forma mais próxima ao que o ser humano está habituado. Posteriormente, o código escrito é interpretado ou compilado. Se for interpretado, a conversão ocorre em tempo real, linha a linha. E se for compilado, o código é alterado, sendo transformado em um objeto de linguagem que o computador entende.

Uma vez que existem diferentes necessidades na programação, cada linguagem possui características próprias, tornando-as mais ou menos indicadas para cada caso e para cada paradigma de programação.
